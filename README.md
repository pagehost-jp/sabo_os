# SABO OS 1.0

## 🧠 プロジェクト概要

**SABO OS 1.0** は、サボ専用の「脳OS」Webアプリです。

### コンセプト
- **サボは「思いついたことを投げるだけ」でOK**
- 整理・分類・タスク化はすべてアプリ（OS）が自動でやる
- あとから見返しても意味がわかるように、アプリが自動で要約

### なぜ作ったか
サボ本人は：
- ジャンル分け・タスク整理・優先順位決めが大の苦手
- 「書く / 話す」だけで、脳の負荷を限界まで減らしたい
- 複雑な管理はアプリ（OS側）に任せたい

## 🚀 セットアップ方法

### 必要なもの
- Node.js (v18以上推奨)
- npm または yarn

### インストール
```bash
# リポジトリをクローン
git clone https://github.com/pagehost-jp/sabo_os.git
cd sabo_os

# 依存関係をインストール
npm install
```

### 起動方法
```bash
# 開発サーバー起動
npm run dev
```

ブラウザで `http://localhost:5173` を開く

### ビルド
```bash
# 本番用ビルド
npm run build

# ビルドしたファイルをプレビュー
npm run preview
```

## 📱 主な機能

### 1. 📝 アイデア入力画面（Sabo Capture）
- テキストエリアに思いついたことを入力
- 送信ボタンを押すだけ
- カテゴリ分類・要約は自動

### 2. ✅ 今日やること画面（Single Task View）★メイン画面
- 「今日やる1つのタスク」だけを大きく表示
- 複数タスクを一度に見せない（脳パンク防止）
- 完了 / あとで の2択だけ

### 3. 📂 リスト画面（管理・振り返り用）
- すべてのアイテムを一覧表示
- フィルタ：すべて / タスク / 完了
- デバッグ・振り返り用

---

## 🤖 自動分類の仕組み

v1.0ではルールベースで分類（将来LLM API連携予定）

### カテゴリ分類
- **task（タスク）**: 「やりたい」「やらなきゃ」などのキーワード
- **idea（アイデア）**: 「ひらめいた」「思いついた」など
- **emotion（感情）**: 「疲れた」「嬉しい」「しんどい」など
- **life（日常）**: 「買い物」「掃除」「ご飯」など
- **system（OS関連）**: 「仕様」「設計」「要件定義」など
- **other（その他）**: 上記以外

### サマリー生成
元の文章を短く要約
- 例: 「アプリの影分身モード作りたい」→「影分身モード機能を実装したい」

## 💾 データ保存

- **v1.0**: localStorage（ブラウザ内ローカル保存）
- **v2.0以降**: Firebase / Supabase などクラウド同期対応予定

データアクセス層は `services/dataService.ts` に分離されているため、
将来的にバックエンドを変更しても影響が少ない設計
